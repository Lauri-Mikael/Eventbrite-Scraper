1. Install Python > 3.10
2. Install necessary modules.
pip install bs4 pandas
3. Run main.py
python main.py
4. Input the city.
The city name should be in lowercase and all the spaces should be replaced with dash(-). 
e.g. Port Isabel -> port-isabel
Some city names should be checked on Eventbrite. You can check this name in the search url.
e.g. Washington D.C -> dc--washington
5. Press Enter and you will get the images and xlsx.
